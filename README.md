# Python-Movie-Correlation

Project dataset: https://www.kaggle.com/datasets/danielgrijalvas/movies

Project objective: To explore, clean & demonstrate correlation between variables. 

1. Python packages & the dataset were imported to Jupyter Notebook
2. Examined the data & performed a number of cleaning exercises in order to prep it for analysis. 
3. Set initial hypothesis to give a direction to the project. 
4. Produced a scatter plot & regression plot using two attributes. 
5. Determined correlation between numberic and non-numberic attributes using pearson, kendall & spearman methods.
6. Developed a correlation matrix for both attribute types. 
7. Used unstack to compair correlation between variables.
8. Examined these results to draw conclusions and to test them against the hypothesis.
